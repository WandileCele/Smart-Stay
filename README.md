# Smart-Stay rec

Smart-Stay is a property booking system developed using ASP.NET Core MVC and SQL Server.

## Features

- Search properties
- View featured properties
- Property details
- Booking system
- User authentication (coming soon)

## Technologies Used

- ASP.NET Core MVC
- C#
- SQL Server
- Entity Framework Core
- HTML
- CSS
- Bootstrap
- Git & GitHub

## Project Structure

Smart-Stay
│
├── Controllers
├── Models
├── Views
├── wwwroot
├── Program.cs
└── appsettings.json

## Getting Started

1. Clone the repository

git clone [https://github.com/yourusername/Smart-Stay.git](https://github.com/WandileCele/Smart-Stay.git)

2. Open the solution in Visual Studio.

3. Restore NuGet packages.

4. Update the database connection string if necessary.

5. Run the project.
